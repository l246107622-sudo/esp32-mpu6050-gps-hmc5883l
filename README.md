# ESP32-MPU6050-GPS-HMC5883L

本项目基于 ESP32 开发板，结合 MPU6050 惯性传感器、GPS 模块和 HMC5883L 磁力计，实现多传感器数据采集、处理与实验结果分析。

## 文件说明

- `firmware.zip`：ESP32 / MicroPython 相关代码文件
- `data.zip`：实验采集到的原始数据文件，包括 GPS、MPU6050、磁力计等数据
- `figures.zip`：实验结果图，包括 GPS 轨迹图、Allan 方差图、磁力计三维效果图等
- `docs.zip`：课程论文、实验说明或相关文档

## 项目内容

本项目主要完成以下功能：

1. ESP32 与传感器模块的连接与数据读取；
2. MPU6050 加速度计和陀螺仪数据采集；
3. GPS 模块定位数据采集与轨迹绘制；
4. HMC5883L 磁力计数据采集与干扰分析；
5. 利用 Python 对实验数据进行可视化分析。
